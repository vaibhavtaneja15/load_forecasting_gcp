Project Overview

This load forecasting system implements an Artificial Neural Network (ANN) built with PyTorch framework using 90 hidden neurons and 11 carefully engineered input features. The model was developed as part of a B.Tech project at the Department of Electrical Engineering, University of Delhi.

Stage 3 Achievement: Complete transition from MATLAB to pure Python implementation, deployed on Google Cloud Platform using Cloud Run for serverless hosting, Vertex AI for model training, BigQuery for data warehousing, and Cloud Storage for model persistence. This architecture enables scalable, production-grade deployment with no proprietary software dependencies.

Technology Stack

Machine Learning & Backend

  • PyTorch: Deep learning framework for ANN implementation
  • Flask: Web framework for RESTful API endpoints
  • NumPy & Pandas: Data preprocessing and numerical computation
  • Scikit-learn: Model evaluation metrics (MSE, R²)

Cloud Infrastructure (GCP)

  • Cloud Run: Serverless container deployment with auto-scaling
  • Vertex AI: Model training, versioning, and hyperparameter tuning
  • BigQuery: Time series data warehousing and analytics
  • Cloud Storage: Model checkpoint and dataset repository
  • Cloud Build: CI/CD pipeline automation
  • Cloud Logging: Monitoring, debugging, and observability

Frontend & Visualization

  • HTML/CSS/JavaScript: Responsive web interface
  • Chart.js: Real-time load trend visualization
  • Docker: Containerization for consistent deployment
