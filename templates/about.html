About the Project

Python-Based ANN for Electrical Load Forecasting with GCP Integration

Project Information

This project titled "Implementation and Deployment of ANN-Based Electrical Load Forecasting System" represents a complete evolution from research to production-ready deployment. The system utilizes a Python-based Artificial Neural Network (ANN) built with PyTorch for predicting hourly electrical load with high precision.

The forecasting model leverages historical load data combined with meteorological and temporal features such as temperature, humidity, month, day, and previous load values. The entire system is deployed on Google Cloud Platform (GCP), utilizing modern cloud services for scalability, reliability, and intelligent data management.

Project Development Phases

Stage 1: Model Development & Training (MATLAB)

  • Dataset collection: 6999 hourly samples over 10 months
  • Data preprocessing using moving mean smoothing and normalization
  • Initial ANN architecture design with 10 input features and 90 hidden neurons
  • Training using Levenberg-Marquardt (LM), Bayesian Regularization (BR), and Scaled Conjugate Gradient (SCG)
  • Model evaluation achieving MSE of 0.0013 and R² of 0.985 with LM algorithm

Stage 2: Hybrid Deployment (MATLAB + Python)

  • Python Flask backend development for API endpoints
  • MATLAB Engine API integration for real-time inference
  • Frontend dashboard creation for user interaction
  • Identification of deployment limitations due to MATLAB licensing dependency

Stage 3: Full Python & Cloud Deployment (Current)

  • Complete ANN reimplementation in Python using PyTorch framework
  • Enhanced feature engineering with 11 input features including previous load dependency
  • Elimination of MATLAB dependency for fully open-source deployment
  • Docker containerization for consistent cross-platform deployment
  • Google Cloud Run deployment for serverless, auto-scaling infrastructure
  • Integration with GCP services: Vertex AI, BigQuery, Cloud Storage, Cloud Build, Cloud Logging
  • Production-ready system with cloud-native architecture

Key Achievements

  • Successfully migrated from MATLAB to pure Python PyTorch implementation
  • Maintained research-grade prediction accuracy (MSE < 0.002, R² > 0.98)

Google Cloud Platform Integration

Vertex AI - Model Training & Management

Vertex AI provides cloud-based infrastructure for training, evaluating, and versioning ANN models. This enables large-scale experimentation, automated retraining pipelines, and structured model lifecycle management without local compute limitations.

BigQuery - Time Series Data Warehouse

BigQuery serves as a centralized data warehouse for storing historical load data, weather parameters, and prediction results. It enables efficient analysis, SQL-based querying, and data-driven model improvements through large-scale analytics.

Cloud Storage - Model & Dataset Repository

Cloud Storage provides scalable and secure storage for trained PyTorch models, Excel datasets, and checkpoint files. It allows seamless access across Cloud Run and Vertex AI services.

Cloud Run - Serverless Deployment

Cloud Run hosts the containerized Flask application, providing automatic scaling, high availability, and pay-per-use pricing. The system handles variable traffic without manual infrastructure management.

Cloud Build - CI/CD Pipeline

Cloud Build automates Docker image creation and deployment, enabling continuous integration and faster, error-free updates to the application with version control integration.

Cloud Logging - Monitoring & Debugging

Cloud Logging provides centralized monitoring, debugging, and performance tracking. It captures prediction requests, errors, and system metrics to improve reliability in production.
